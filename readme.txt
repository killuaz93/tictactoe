Access Application Via :

File  (using in-browser babel transformer)
------------------------------------------
Steps : 
1. Double click index.html file in this folder
example : file:///Users/krisnosotanto/react/tictactoe/index.html

Webserver PHP XAMPP (using precompiled babel script)
----------------------------------------------------
Steps : 
1. Start XAMPP Apache Service
2. Copy this folder to htdocs
3. Access http://localhost/tictactoe/index.php
